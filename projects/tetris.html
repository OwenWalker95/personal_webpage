Overview

Started around March 2025. 'Finished' June 2026

After numerous small projects in C which never materialised into anything, I started making a simple Tetris clone. I decided this was something I could actually bring to some sort of completion without spending too long on it.

There are still improvements I could make, mostly smoothness of the animations and aesthetic improvements. Some sort of hi-score system. And the code itself could do with tidying up.

I may revisit it at some point, but the aim of this was just to get a sense of 'completion'.

This was developed on linux for windows subsystem, has not been tested on any other machine.

Key features

Functioning Tetris Game, with a start screen, game over screen and exit button

Game keeps score and increases in speed as score is gained

Game can be restarted from game over screen

Animations for clearing lines and for game ending
