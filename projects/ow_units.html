About

Overview

Started sometime in 2024. This is a 'library' of C functions and data structures I've built up through doing my side projects while learning C.

As I wanted to focus on learning the language rather than learning various libraries and having things abstraced away, I coded everything from scratch besides the lower level input/output operations. For those I utilised the SDL library

Still very much a work in progress and something I will add to if I continue working in C.

Key features

  • Display functions for opening a window, drawing basic geometric shapes, and rendering text and .png images
  • Input functions for taking mouse/keyboard input when using a window. As well as CLI input parsing.
  • String handling functions (currently only one, for converting a string to a float).
  • Functions for getting timestamps and a timer.
  • Some general utilities such as random number generator.
